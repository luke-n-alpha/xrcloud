# Work Logs Naming Convention

이 디렉토리의 모든 작업 로그 문서는 다음 명명 규칙을 따릅니다.

## 형식

`YYYYMMDD-일련번호-주제.md`

-   **YYYYMMDD**: 문서 생성 또는 작업 날짜 (예: `20260108`)
-   **일련번호**: 해당 날짜에 생성된 문서의 순서를 나타내는 3자리 숫자 (예: `000`, `001`)
-   **주제**: 문서의 핵심 내용을 간결하게 설명하는 영문 소문자 (예: `comprehensive-cleanup-plan`)

## 예시

-   `20260108-000-comprehensive-cleanup-plan.md`
-   `20260108-001-daily-log.md`
